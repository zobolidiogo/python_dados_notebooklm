## O Desafio

Criar um **miniguia de estudo estruturado** utilizando IA Generativa como apoio ao aprendizado, seguindo as etapas:

- Definição de um tema relevante para desenvolvimento profissional
- Curadoria de fontes técnicas abertas e confiáveis
- Organização do conhecimento no NotebookLM
- Elaboração de perguntas e prompts exploratórios
- Geração de resumos estruturados e glossário
- Documentação completa do processo
- Publicação em repositório GitHub
